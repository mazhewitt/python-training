# Comparison & Boolean
- Comparison operators:
    - "== equal to"
    - "!= not equal to"
    - "< less than"
    - "> greater than"
    - "<= less than or equal"
    - ">= greater than or equal"
- = is the assignment operator, == is a comparison operator
- Python syntax: **if** keyword, condition, colon, indented block of code starting from next line.
