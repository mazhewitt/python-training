# Conditional Operators**

| Operator | Definition       |
|----------|------------------|
| **>**        | Greater          |
| **>=**       | Greater or equal |
| **<**        | Lower            |
| **<=**       | Lower or equal   |
| **!=**       | Not equal        |
| **==**       | Equal            |

```
younger_brother = 18
elder_brother = 22
print(younger_brother < elder_brother)

> True
```
