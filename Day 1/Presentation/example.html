It's pretty simple to code

but to run it locally you need Node.js

Who want's to store PowerPoint files in GitHub anyway?

  • To get Node.js download the latest from devtools
  • Install it using nvm
  • Clone this repo
  • you need the ubs assets and the reveal folders
  • run : in the reveal folder
    
                    $> nmp install
                    $> npm start
  • Edit the index.html for the title and add your slides
  • edit the UBS.scss file to add new styles - grunt will automatically create the UBS.css file
You can also include speaker notes
