# Let's begin with print()
- Bult-in function
- Prints the values to standard output (console)
- Values are seperated by space by default (sep)
- After the last value newline symbol \n is appended by default (end)
```
    print('Hello', 'world')
    >    Hello world
    print('Hello', 'world', sep='_')
    >    Hello_world
    print('Hello', end=' ')
    print('world')
    >    Hello world
```
